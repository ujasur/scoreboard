Basic rules

Team consist of the master and many voters. Only voters can participate in voting.
All usernames must be unique. Password for each user is his username (for simplicity).
Initially team has a master user with password master, who can add team members.

Permissions
Master
  • Add/Remove users
  • Open session
  • Close session
  • Reset session
  • See voters' scores
Voter
  • Open session
  • Close session
  • Reset session
  • Vote in session
Session rules
Open User who opened a session becomes leader ★
Reset Only session leader is allowed
Vote Only voter is allowed
Close If any of the following is true:
  • If you are master (regardless of session leader).
  • If you are leader.
  • Any user after [[index .Data "leaderLife"]] hour of leader inactivity
